# Gabriela Nunes — Payload + Next.js

Scaffold inicial do projeto: Payload CMS v3 + Next.js App Router.

## Comandos

```bash
npm install
npm run dev
npm run lint
npm run build
```

## Variáveis de ambiente

- `DATABASE_URI`
- `MONGODB_URI`
- `PAYLOAD_SECRET`
- `NEXT_PUBLIC_SITE_URL`
- `NEXT_PUBLIC_ANALYTICS_ID`

## Observações

- O app está preparado para MongoDB Atlas.
- O deploy principal recomendado é Vercel.
- Netlify e HostGator exigem cuidado extra com a parte do backend/admin.
