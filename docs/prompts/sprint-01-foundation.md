## sprint-01-foundation

Você é um desenvolvedor sênior responsável por criar a fundação de um projeto de produção.

Objetivo:
Criar um projeto executável usando:

- Next.js latest App Router
- TypeScript
- Payload CMS v3
- MongoDB Atlas
- Tailwind CSS

Requisitos obrigatórios:

1. configurar Payload v3 corretamente;
2. criar estrutura de pastas exatamente conforme arquitetura;
3. configurar aliases TS;
4. configurar ESLint;
5. configurar Prettier;
6. criar design tokens;
7. criar tema base;
8. configurar env validation com Zod;
9. preparar Vercel deploy;
10. adicionar .env.example.

Critérios de aceite:

- npm run dev executa sem erro;
- admin do Payload abre;
- build executa;
- sem warnings críticos;
- TypeScript sem any desnecessário.

Não invente arquitetura.
Siga estritamente este documento.
